# Цель проекта:

```
1. Создать сайт-визитку используя основные команды разметки страницы и 
специальные семантические элементы.
2. Сделать страницу "рабочей". По ссылкам должен осуществляться переход.
3. Работа должна включать в себя пояснительные комментарии.
```

# Задачи выполненные в проекте:

```
1. Заполнена главная веб-страница __index.html.___
2. Заполнена страница __about.html.__
3. Заполнена страница __contacts.html.__
4. Ссылки на страницы работают, по ним осуществляется переход.
5. Текстовое содержимое заключено в теги.
6. Добавлены изображения тегом __<img>__.
7. Сайт проверен через __Validator W3C__ и не имеет ошибок.
8. Сайт содержит пояснительные комментарии.
```

# Планы запланированные на будущие веб-страницы и веб-сайты:

```
1. Использовать более углубленные команды разметки страница __HTML__ и так же включать
визуальное представление с помощью каскадных таблиц стилей  __CSS__.
2. Включать в работу __фреймворки__ и __библиотеки__ для оптимизации и автоматизации работы.
```
